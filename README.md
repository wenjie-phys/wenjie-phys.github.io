# Wenjie Ji Personal Homepage

This is a Hugo site using the PaperMod theme.

## Local preview

Install Hugo Extended 0.146.0 or newer, then fetch the theme:

```sh
git clone --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
hugo server
```

The GitHub Actions workflow fetches PaperMod automatically before building the site for GitHub Pages.
